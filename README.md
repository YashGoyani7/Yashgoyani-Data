# Netflix Data Analysis Project

## Objective
The objective of this project is to analyze Netflix movies and TV shows data
to identify trends based on content type, release year, country, genre, and rating.
This project uses Python for data cleaning, analysis, and visualization.

## Key Questions
- How many movies and TV shows are available on Netflix?
- Which countries produce the most Netflix content?
- How has Netflix content grown over the years?
- Which genres dominate Netflix?
- What are the most common content ratings?

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI (optional)
